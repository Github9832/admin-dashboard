## 🚀 Features

### 🎨 UI / UX
- Modern **glassmorphism interface**
- Clean sidebar navigation  
- Smooth layout with soft edges & premium blue theme  
- Fully responsive for all screen sizes  
- Beautiful card designs and metric displays  

### 📄 Includes Multiple Pages
- 🏠 **Dashboard (Home)**  
- 📊 **Analytics Page**  
- 👤 **Users Page**  
- ⚙ **Settings Page**  
- 📁 **Reports Page**  
- 🔐 **Login Page**  

### 🧩 Components
- User cards + user table  
- Analytics metric cards  
- Report cards + report history table  
- Fully styled form elements  
- Pure CSS toggles and switches  
- No JavaScript required
